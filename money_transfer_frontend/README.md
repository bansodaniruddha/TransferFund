# Money Transfer Frontend (React)

This is a simple React frontend for the Money Transfer OTP Spring Boot backend.

Assumptions:
- Backend is running at http://localhost:8080
- OTP is a dummy OTP returned by the backend in the transfer response or visible via API.

How to run:
1. Install Node (>=16) and npm.
2. From this folder, run:
   ```bash
   npm install
   npm start
   ```
3. The app will open at http://localhost:3000
